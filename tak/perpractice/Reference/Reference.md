# 참조자(Reference)
- 할당된 **하나의 메모리 공간**에 **둘 이상의 이름**을 부여하는 역할을 해준다.
## 참조자의 역할
- *이미 선언된* 변수 앞에 올 경우
    - **주소 값의 반환을 명령**하는 역할이다.
- *새로 선언되는* 변수 앞에 올 경우
    - **참조자의 선언**을 하는 역할이다.
## 변수와 참조자
```plaintext
변수와 참조자는 선언 방식에서 확실한 차이를 보이지만, 참조자가 선언이 되고 나면 변수와 차이가 없다.
```
