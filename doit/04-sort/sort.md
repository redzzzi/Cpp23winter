# 버블 정렬
- 정의: 두 인접한 데이터의 크기를 비교해 정렬하는 방법
- 시간 복잡도: **O(n<sup>2</sup>)**

# 선택 정렬
- 정의: 대상 데이터에서 *최솟값 또는 최댓값을 찾고*, 남은 정렬 부분의 *가장 앞에 있는 데이터와 swap*하는 정렬 방법
- 선택 정렬 자체를 묻는 코테 문제는 잘 없다고 함.
